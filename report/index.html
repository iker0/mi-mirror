How Do Vision Models Render Reflections in Mirrors?

Searching for "Reflection Heads" in Flux — a mechanistic interpretability investigation

Mechanistic Interpretability · Diffusion Models · Flux

Large language models contain induction heads — attention heads that learn to copy patterns from earlier in the sequence. Could image generation models contain analogous reflection heads — circuits that specifically handle rendering reflections in mirrors?

We investigate this question in Flux-schnell, a state-of-the-art text-to-image model built on the MMDiT (Multi-Modal Diffusion Transformer) architecture. By extracting and analyzing attention patterns across all 1,368 attention heads (57 blocks × 24 heads), we search for heads that preferentially attend across the mirror boundary — from reflection regions to object regions — when generating mirror images but not matched non-mirror scenes.

The Architecture

Flux uses a hybrid transformer with two types of blocks:

Input: text tokens (T5-XXL) + image tokens (32×32 grid) Dual-Stream Blocks (0-18) — 19 blocks → Separate Q/K/V projections for text and image → Concatenate into joint sequence for attention → Split apart for separate FFN streams Single-Stream Blocks (19-56) — 38 blocks → Text + image concatenated as single sequence → Unified self-attention + MLP Each block: 24 heads × 128-dim = 3072 hidden dim QK-Norm (RMSNorm) + Rotary Position Embeddings

At 512×512 resolution, Flux produces a 32×32 grid of image tokens (1,024 total, due to 2×2 patchification + 8× VAE downsampling). Each attention head operates on a joint sequence of ~512 text + 1024 image = ~1536 tokens.

Method

Step 1 Selectivity Screening

Generate mirror and non-mirror image pairs. For each head, compute Cross-Region Attention (CRA) — how much the reflection region attends to the object region. Heads with high CRAmirror - CRAnonmirror are candidates.

Step 2 Entropy Analysis

True reflection heads should show lower entropy on mirror images (focused cross-region attention) vs. diffuse attention on non-mirror images. Combine with selectivity into a HIES score.

Step 3 Temporal Profiling

Track CRA across denoising timesteps. When does reflection attention peak? Early steps (layout) vs. late steps (details) tells us about the reflection circuit's role.

Step 4 Spatial Patterns

Visualize the full attention maps for top candidates. Do reflection-region queries attend to spatially corresponding object tokens? This is the "smoking gun" for reflection heads.

Metrics

Cross-Region Attention: CRA(h, l) = meani∈Rref, j∈Robj Aij
Selectivity Score: S(h, l) = CRAmirror(h, l) - CRAnonmirror(h, l)
Entropy: H(h, l) = -∑ij A · log(A)
HIES: S × (Hnonmirror - Hmirror) / Hmax

Step 2 Entropy Analysis

For the top candidate heads, we extract full attention maps and compute entropy. Reflection heads should exhibit lower entropy on mirror images — meaning their attention is more focused and structured, rather than diffuse.

Step 3 Temporal Profile

Flux-schnell uses only 4 denoising steps (a distilled schedule). We track CRA across these steps to understand when reflection attention is most active. Does the model establish mirror correspondence early (layout phase) or refine it later (detail phase)?

Step 4 Spatial Attention Maps

The most direct evidence: for candidate reflection heads, we visualize where reflection-region tokens attend. If a head truly implements reflection, queries from the right side of the image (mirror) should strongly attend to corresponding positions on the left side (object).

Discussion

Our analysis pipeline — selectivity screening, entropy analysis, temporal profiling, and spatial visualization — provides a systematic framework for identifying reflection-specialized attention heads in diffusion transformers. The approach is analogous to how induction heads were discovered in LLMs: by defining a behavioral signature (cross-region attention) and searching across all heads.

Limitations & Next Steps

Causal validation (Steps 5-6) remains essential: selectivity is correlational. Zero-ablation experiments would test whether candidate heads are necessary for reflection rendering, and circuit analysis could reveal multi-head composition (e.g., one head identifying the mirror boundary, another copying features across it).

ROI sensitivity: We use a simple vertical split, which may not perfectly align with actual object/reflection boundaries. Bounding-box ROIs or segmentation masks would improve precision.

NF4 quantization: While absolute attention values may shift under 4-bit quantization, selectivity (relative differences) should be robust. Verification with full-precision inference on A100 is recommended.
